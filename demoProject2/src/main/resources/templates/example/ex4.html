삼항 연산자

		- 타임리프 속성(th:)에 삼항연산자 작성가능
		조건식 ? true인경우 : false인경우

Elvis ( ?: )

		[작성법]
		값A   ?:   값이 없을때
		
		- 좌측과 우측으로 나눌 때, 
		- 좌측에 값이 있으면 좌측이, 없으면 우측이 출력.

No-Operation 연산자 ( 값 ?: _)

		- 조건식의 값이 없을 경우 (==null)
		타임리프 코드를 해석하지 않는 연산자
		
		- 타임리프 코드 해석하지 않는다는 것은
		-> 일반 HTML 태그로 동작
		-> 따라서, th:block 같은 타임리프 태그는 안된다는 거고, html 태그만 된다는 거임 .
		-> 조건식에 값이 없을 경우 html 태그 여는 태그와 닫는태그 사이에 있는 텍스트가 출력됨.
